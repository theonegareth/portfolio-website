# Portfolio Website

A personal portfolio website to showcase projects, experience and contact information. This repository contains the source for a fast, responsive, and accessible portfolio site.

## Features
- Project gallery with links and screenshots
- About / resume section
- Contact form (or mailto link)
- Responsive layout for mobile and desktop
- Easy to customize and deploy

## Built with
- HTML, CSS, JavaScript (or replace with your framework: React / Vue / Svelte / Next.js / Gatsby)
- Optional: Vite / Create React App / Parcel for dev tooling

## Getting started

### Prerequisites
- Node.js and npm or yarn (if using a JS framework)
- Git (optional)

### Install
Clone the repo and install dependencies:
```bash
git clone <repo-url>
cd portfolio-website
# if using npm
npm install
# or if using yarn
yarn
```

### Development
Run the development server (adjust command to your stack):
```bash
npm run dev
# or
npm start
```

### Build
Create a production build:
```bash
npm run build
```

### Deploy
Common options:
- GitHub Pages
- Netlify / Vercel
- Static host on your own domain
